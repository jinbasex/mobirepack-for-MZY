# mobirepack-for-MZY
  一个针对漫自由漫画站，对mobi重新为kpw3 kindle 2022等 对应dpi的设备进行重新打包的软件


## 🛠️如何使用
  下载解包之后
  在 [漫自由](https://mhx12.com/)  下载mobi漫画资源后 直接将整个文件夹或者文件拖到py 或者exe文件即可

  如果使用py需要补充对应的依赖库 

  exe已打包依赖库 可直接使用

 感谢 [书伴](https://bookfere.com/post/92.html#kg_1)  提供的kindlegen文件
